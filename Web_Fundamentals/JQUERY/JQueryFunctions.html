.click()

Click button to increase count.

.hide()

Click button to hide itself.

.show()

Click button to show some hidden text.

This text is hidden

.toggle()

Click button to see title toggled between showing and hidden.

.slideDown()

Click button to see button slide down.

.slideUp()

Click button to see it slide up.

.slideToggle()

Click button to see the text perform a slideToggle.

.fadeIn()

Click button to see the hidden text fade in.

This text will fade in

.fadeOut()

Click button to see the button fade out.

.addClass()

Click button to see a class get added which changes the button color.

.before()

Click button to see another button get added before the current button.

.after()

Click button to see another button get added after the current button.

.append()

Click button to see another button get appended to the current button.

.html()

Click button to see the text change to the buttons html.

.attr()

Click button to see the text change to the buttons "checked" attribute.
